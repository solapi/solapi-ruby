
### 그룹 발송 예제
```
create_group.rb               그룹 생성
get_group_info.rb             그룹 정보 조회
delete_group.rb               그룹 삭제
get_group_list.rb             그룹 목록 조회

add_group_message.rb          그룹 메시지 추가
delete_group_messages.rb      그룹 메시지 삭제
get_group_messages.rb         그룹 메시지 조회

send_group_message.rb         그룹 메시지 발송
send_group_msg_with_image.rb  그룹 메시지(이미지) 발송
```
